## 附录A：开发环境要求

### 必需软件

| 软件 | 最低版本 |
|------|----------|
| Node.js | 22 LTS |
| pnpm | 9.x |
| Docker | 24+ |
| Git | 2.40+ |
| VS Code | 最新版 |

### 推荐 VS Code 插件

- Tailwind CSS IntelliSense
- Biome
- Drizzle ORM
- Thunder Client
- Error Lens

### 本地开发启动

```bash
# 1. 克隆项目
git clone <repo-url>
cd BetterWrite

# 2. 安装依赖
pnpm install

# 3. 启动基础设施
docker compose up -d

# 4. 数据库迁移
pnpm db:migrate
pnpm db:seed

# 5. 启动开发服务器
pnpm dev
# 同时启动: Web (localhost:3000) + Worker + API

# 6. 启动移动端（可选）
pnpm dev:mobile
```

## 附录B：风险与缓解

| 风险 | 影响 | 概率 | 缓解措施 |
|------|------|------|----------|
| AI API 不稳定 | 高 | 中 | 多Provider路由 + 降级策略 |
| AI 批改准确度不足 | 中 | 中 | 教师人工复核 + 持续优化 Prompt |
| 移动端性能问题 | 中 | 中 | React Native 性能优化 + 原生模块 |
| 深圳中考评分标准变化 | 中 | 低 | 评分标准可配置化 |
| 数据安全合规 | 高 | 低 | 加密存储 + 权限控制 + 审计日志 |
| 用户量增长超出预期 | 中 | 中 | 弹性架构 + 自动扩缩容 |
