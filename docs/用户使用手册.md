## 1. 系统概述

### 1.1 系统简介

BetterWrite 是一款专为深圳中考英语作文设计的 AI 智能辅导系统，服务于教师、学生和学校管理员三类用户。系统提供作文任务管理、AI 智能批改、五维能力分析、错题本、AI 写作助手、题库练习等核心功能，覆盖英语写作教学的全流程。

### 1.2 核心功能

| 功能模块 | 学生 | 教师 | 学校管理员 | 超级管理员 |
|---------|:---:|:---:|:---------:|:---------:|
| 作文写作与提交 | ✓ | - | - | - |
| AI 智能批改 | ✓ | ✓ | ✓ | ✓ |
| 作文任务管理 | ✓ | ✓ | - | - |
| 班级管理 | - | ✓ | ✓ | ✓（全局） |
| 学情分析 | - | ✓ | ✓ | ✓ |
| 写作成长报告 | ✓ | - | - | - |
| 错题本 | ✓ | - | - | - |
| 题库练习 | ✓ | - | - | - |
| AI 写作助手 | ✓ | - | - | - |
| 教师复核 | - | ✓ | ✓ | ✓ |
| 学校管理 | - | - | ✓ | ✓ |
| 系统管理 | - | - | - | ✓ |

### 1.3 评分体系

系统采用 **深圳中考英语作文评分标准（15分制）**，参考真实阅卷的“定档制”思路，从五个维度评估作文：

| 维度 | 满分 | 说明 |
|------|:---:|------|
| 内容 | 5.0 | 要点是否齐全、展开是否充分、是否紧密切题 |
| 语言 | 4.0 | 语法是否正确、词汇是否丰富、表达是否得体 |
| 结构 | 2.5 | 文章结构是否清晰、逻辑是否连贯 |
| 审题扣题 | 2.0 | 是否准确理解题目要求、文体格式、人称时态 |
| 卷面 | 1.5 | 字数是否达标、格式是否规范 |

**评分逻辑：** 先根据内容要点、语言准确性、结构清晰度综合判定档次，再在同档内微调各维度得分。遗漏核心要点会显著扣内容分，甚至直接降档。

**总分档次对照：**

| 档次 | 分数 | 标准 |
|:----:|:----:|------|
| 第一档（优） | 15-13 | 要点齐全且展开充分，语言正确无误，行文流畅，结构清晰 |
| 第二档（良） | 12-10 | 要点较完整，语言有少量错误，意思较连贯，结构基本清晰 |
| 第三档（中） | 9-7 | 要点部分缺失或展开不足，有一些语言错误，连贯性一般 |
| 第四档（及格） | 6-4 | 要点缺失较多，语言错误较多，意思表达部分清楚 |
| 第五档（差） | 3-0 | 只能写出少量相关词，不能表达完整意思 |

## 2. 快速入门

### 2.1 访问系统

- **Web 端**：通过浏览器访问系统网址（如 `http://localhost:3000`）
- **移动端**：通过 Expo Go 扫码或安装 APK 后使用

### 2.2 登录

1. 打开系统首页，点击"登录"按钮
2. 输入邮箱地址和密码
3. 点击"登录"按钮进入系统

### 2.3 默认账号

执行数据库种子命令后，系统会创建以下账号，密码随机生成并在命令行输出：

| 角色 | 邮箱 |
|------|------|
| 超级管理员 | superadmin@betterwrite.cn |
| 学校管理员 | admin@school.com |
| 教师 | teacher@school.com |
| 学生 | student@school.com |

> **注意**：生产环境部署后，请务必修改默认密码。

### 2.4 注册新账号

1. 在登录页面点击"注册"链接
2. 填写姓名、邮箱、密码
3. 选择角色（教师或学生）
4. 可选：填写学校代码和班级代码以自动加入学校和班级
5. 点击"注册"完成

## 3. 学生功能指南

### 3.1 学习首页

登录后进入学生学习首页，可查看：

- **待完成任务**：尚未完成的作文任务数量
- **已批改作文**：已完成批改的作文数量
- **平均得分**：所有已批改作文的平均分
- **每日一句**：英语名言警句展示
- **分数分布图**：已完成作文的分数分布柱状图
- **最近任务**：最新的待完成作文任务列表
- **最近作文**：最近提交的作文列表

### 3.2 作文任务

#### 查看任务列表

1. 左侧导航栏点击"作文任务"
2. 查看所有已发布的作文任务
3. 每个任务卡片显示：体裁标签、标题、要求、字数限制、截止日期、状态

#### 完成作文任务

1. 在任务卡片上点击"开始写作"按钮
2. 进入写作页面，包含：
   - **任务要求面板**：显示题目、要求、要点、字数范围
   - **写作编辑器**：输入作文内容，实时统计字数
   - **计时器**：记录写作时长
   - **自动保存**：草稿每 30 秒自动保存
3. 写作完成后，完成**自查清单**（检查要点覆盖、格式、人称时态等）
4. 点击"提交"按钮完成提交

#### 自由写作

如无指定任务，可点击"自由写作"按钮，自主选择主题写作并提交批改。

### 3.3 查看批改结果

1. 在"我的作文"列表中点击已批改的作文
2. 批改结果页面包含：

   - **总得分**：15 分制评分及档次评定
   - **五维雷达图**：审题扣题、内容、语言、结构、卷面五个维度的能力展示
   - **错误列表**：按类型分类的错误详情，每个错误包含：原文、修正、解释、位置
   - **修改后范文**：AI 优化后的完整作文
   - **改进建议**：按优先级（高/中/低）分类的改进建议
   - **教师复核**：如教师已复核，可查看教师评语和评分

### 3.4 写作成长

1. 左侧导航栏点击"写作成长"
2. 成长报告页面包含：

   - **统计概览**：总作文数、平均分、班级排名、写作水平等级
   - **五维能力雷达图**：审题扣题、内容、语言、结构、卷面能力评估
   - **进步曲线**：历次作文分数变化趋势图
   - **成就勋章**：通过完成特定里程碑获得的成就徽章

### 3.5 错题本

1. 左侧导航栏点击"我的错题本"
2. 错题本按错误类型分组展示，包含：
   - 错误类型标签
   - 总数、未掌握数、已掌握数
   - 最新错误原文和修正
3. 点击错误类型进入详情，查看具体错误列表
4. 点击"同步错题"按钮，从最新批改结果中同步错误
5. 每个错误可标记为"已掌握"或"未掌握"

### 3.6 题库练习

#### 题库练习

1. 左侧导航栏点击"题库练习"
2. 在"题库练习"标签页中：
   - 可按体裁、难度筛选题目
   - 点击题目卡片开始练习
   - 练习完成后提交，获得 AI 即时反馈

#### 限时模拟

1. 在"限时模拟"标签页中：
   - 选择模拟题目
   - 系统自动计时，在规定时间内完成写作
   - 提交后获得 AI 批改和评分

#### 练习历史

1. 在"练习历史"标签页中：
   - 查看所有练习记录
   - 包括得分、用时、题目类型等信息

### 3.7 AI 写作助手

1. 左侧导航栏点击"AI 写作助手"
2. 提供四种辅助模式：

   | 模式 | 功能 | 用法示例 |
   |------|------|---------|
   | **AI 润色** | 优化表达，不改变原意 | 粘贴英文段落，获得润色版本 |
   | **句型升级** | 将简单句升级为高级句式 | 粘贴简单句子，获得高级表达 |
   | **同义词推荐** | 替换基础词汇为高级词汇 | 输入 `important \| It is important to...` |
   | **语法检查** | 检查语法错误并给出修改建议 | 粘贴英文，获得语法修正 |

3. 每次查询结果会保存在历史记录中，方便回顾

### 3.8 提交方式

#### 手动输入

在写作页面直接输入或粘贴作文内容。

#### 拍照识别（OCR）

1. 在移动端可点击拍照按钮
2. 拍摄手写作文照片
3. 系统自动识别图片中的文字并填入编辑器
4. 确认识别结果无误后提交

## 4. 教师功能指南

### 4.1 教师首页

登录后进入教师工作台，可查看：

- **统计概览**：任教班级数、学生总数、待批改作文数、平均分
- **班级列表**：所有任教班级的基本信息（班级名称、年级、学生数）
- **最近任务**：最近创建的任务列表
- **最近作文**：学生最近提交的作文及状态

### 4.2 作文任务管理

#### 创建任务

1. 左侧导航栏点击"作文任务"
2. 点击"布置任务"按钮
3. 填写任务信息：
   - 标题（英文）
   - 体裁（记叙文/议论文/说明文/应用文）
   - 写作要求（英文）
   - 要点提示（3-5 个要点）
   - 目标班级
   - 字数范围（默认 80-125 词）
   - 截止日期（可选）
4. 点击"创建"完成

#### AI 辅助出题

1. 在任务创建页面点击"AI 出题"按钮
2. 输入主题关键词
3. 选择体裁和年级（可选）
4. 系统自动生成：标题、体裁、要求、要点、参考范文
5. 生成的内容可编辑后使用

> **注意**：AI 辅助出题有频率限制，每分钟最多 5 次。

#### 管理任务状态

| 状态 | 说明 | 可执行操作 |
|:----:|------|-----------|
| 草稿 | 刚创建，学生不可见 | 编辑、发布、删除 |
| 已发布 | 学生可见并可提交作文 | 截止、查看提交情况 |
| 已截止 | 学生不可再提交 | 查看提交情况 |

- **发布任务**：点击"发布"按钮，任务对学生可见
- **截止任务**：点击"截止"按钮，学生不再允许提交

### 4.3 批改与复核

#### 查看学生作文

1. 左侧导航栏点击"作文管理"
2. 查看所有学生提交的作文列表
3. 可按班级、状态筛选
4. 每个作文条目显示：学生姓名、标题、状态、得分、提交时间

#### 教师复核

当 AI 批改完成后，教师可进行复核：

1. 点击作文条目进入详情
2. 查看 AI 批改结果（评分、错误、建议）
3. 在"教师复核"区域：
   - 填写评语（可选，最多 2000 字）
   - 给出教师评分（可选，0-100 分）
   - 至少需要提供评语或分数其中之一
4. 点击"保存复核"完成

### 4.4 学情分析

#### 班级分析

1. 左侧导航栏点击"学情分析"
2. 选择要分析的班级
3. 分析页面包含：

   - **班级概览**：学生数、作文数、平均分
   - **平均分趋势**：历次任务的平均分变化图
   - **分数分布**：各分数段人数分布
   - **高频错误 Top10**：班级学生最常见错误类型排行
   - **体裁对比**：不同体裁的平均分对比

#### 学生分析

1. 在学情分析页面切换到"学生分析"标签页
2. 选择学生查看详细分析：

   - **个人统计**：作文数、平均分
   - **五维能力**：审题扣题、内容、语言、结构、卷面能力值
   - **分数趋势**：历次作文分数变化图
   - **错误分布**：各类错误占比
   - **最近作文**：最近提交的作文列表

#### 数据导出

1. 在班级分析页面点击"导出 CSV"
2. 下载包含班级所有作文数据的 CSV 文件
3. 可用 Excel 或其他工具打开分析

### 4.5 学生管理

#### 查看学生列表

1. 左侧导航栏点击"学生管理"
2. 查看所有学生列表
3. 可按班级筛选、按姓名/学号搜索
4. 每个学生显示：姓名、学号、班级、标签、作文数、平均分

#### 学生详情

1. 点击学生进入详情页
2. 查看：班级信息、个人标签、平均分、作文数、最近作文列表

#### 批量导入学生

1. 在学生管理页面点击"导入学生"
2. 选择目标班级
3. 准备 CSV 文件，格式为：
   ```
   name,email,studentno
   张三,zhangsan@school.com,S2024001
   李四,lisi@school.com,S2024002
   ```
4. 上传 CSV 文件
5. 系统自动创建账号，初始密码为随机生成的 8 位密码

### 4.6 教学资源管理

1. 左侧导航栏点击"教学资源"
2. 按资源类型查看：范文、写作技巧、词汇积累、语法讲解等
3. 可查看、创建和管理教学资源
4. 资源支持按体裁、难度筛选

## 8. 常见问题

### 8.1 账号与登录

**Q: 忘记密码怎么办？**  
A: 请联系教师或管理员重置密码。目前系统不支持自助找回密码功能。

**Q: 登录提示"邮箱或密码错误"？**  
A: 请检查邮箱是否输入正确，注意区分大小写。如确认无误，请联系管理员检查账号是否被禁用。

**Q: 如何修改密码？**  
A: 登录后，在个人设置页面可修改密码。修改密码有频率限制，每分钟最多 5 次。

### 8.2 作文写作

**Q: 作文提交后多久能拿到批改结果？**  
A: 通常在 1-5 分钟内完成批改，具体时间取决于作文长度和系统负载。如长时间未完成，请联系管理员检查 Worker 服务状态。

**Q: 提交后还可以修改作文吗？**  
A: 提交后不可修改。如需重新提交，建议在提交前仔细检查。可在自由写作中重新写一篇。

**Q: 为什么我的作文状态一直显示"批改中"？**  
A: 可能原因：AI 服务正在处理中（正常等待 1-5 分钟）；或 AI 服务配置异常，请联系管理员检查。

**Q: 字数不够会有什么影响？**  
A: 按照深圳中考评分标准，词数不足 80 词将降一档评分。建议控制在 80-125 词之间。

### 8.3 AI 批改

**Q: AI 批改的准确率如何？**  
A: AI 批改基于深圳中考评分标准，从五个维度进行评估。但 AI 不能完全替代人工判断，建议教师进行复核。

**Q: 教师复核和 AI 批改有什么区别？**  
A: AI 批改是系统自动完成的初步批改，教师复核是教师对 AI 批改结果的人工审核和修正。最终成绩以教师复核为准。

**Q: 为什么没有 AI 批改结果？**  
A: 可能原因：系统未配置 AI 供应商密钥（AI Key），此时会使用模拟评分；或 AI 服务调用失败。

### 8.4 其他

**Q: 不同浏览器有兼容性问题吗？**  
A: 建议使用最新版 Chrome、Edge、Firefox 或 Safari 浏览器。

**Q: 移动端和 Web 端数据同步吗？**  
A: 是的，移动端和 Web 端共享同一后端服务，数据实时同步。

**Q: 如何查看历史批改记录？**  
A: 在"我的作文"列表中可查看所有历史作文和批改结果。

**Q: 支持哪些 AI 供应商？**  
A: 系统支持 OpenAI、DeepSeek、Anthropic 三家 AI 供应商，配置其中一个即可使用 AI 批改功能。
